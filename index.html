The Journey

The Construction of this game began with the HTML code that incorporated Canvas element into it.

Followed by the actual building of the game that was done in Javascript and with the assistance of Phaser 3.10.1 "Hayashi".

Finally this webage was made using Bootstrap and styled using CSS.

The Game

The Player must reach the Treasure while avoiding all fireballs shot by the Dragon

The Player has three live which decrease by 1 for every hit

A Health kit is available that restores one life

The trees and plants block the fireballs
